# UNIR Smart Cell Analyzer

Aplicación de escritorio en Python para cargar imágenes médicas, visualizar regiones de interés (ROI) y apoyar el análisis de células musculares.

## Tecnologías usadas
- Python
- Tkinter
- ttkbootstrap
- OpenCV
- NumPy
- Pillow
- ReportLab

## Instalación

pip install -r requirements.txt
